# 25-js-SpamFilter
Сделайте интерфейс для анонимных комментариев со спамфильтром. 
Необходимо реализовать отображение и добавление сообщений, а также функцию `checkSpam(str)`, заменяющую `'viagra'` или `'XXX'` на `***`
